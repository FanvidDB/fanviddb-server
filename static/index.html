FanvidDB

A central repository for fanvid-related metadata, useful for Plex or any other purpose.

Hello world! Welcome to FanvidDB. This is a project to create an index of fanvids. One of the primary use cases is to provide integration with Plex so that people's personal fanvid libraries can be consumed as easily as their TV shows or movies.
